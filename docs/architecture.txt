Архитектура проекта "Chatbot Interface":

1. Frontend:
   - index.html: Основной HTML-шаблон с адаптивным дизайном.
   - style.css: Стили для оформления, анимаций и синтаксической подсветки.
   - script.js: Логика обработки пользовательского ввода, распознавания специальных команд (terminal, view_file, edit_file) и взаимодействия с сервером.

2. Backend:
   - server.py: Flask сервер, принимающий запросы от фронтенда, маршрутизирующий их и взаимодействующий с моделью Gemini.
   - gemini_client.py: Модуль для обращения к Gemini API (модель gemini-2.0-flash-thinking-exp) с использованием предоставленного API ключа.

3. Документация и инструкции:
   - README.md: Общая информация о проекте и инструкция по запуску.
   - docs/architecture.txt: Детальное описание архитектуры проекта.
   - instructions/bot_instructions.txt: Инструкции для бота по выполнению и обработке команд.

4. Принцип работы:
   - Пользователь вводит сообщение через адаптивный интерфейс.
   - Если сообщение содержит специальную команду, скрипт на фронтенде обрабатывает её (с анимацией и подсветкой) и отправляет запрос на сервер.
   - Сервер передает сообщение в модель Gemini, получает ответ и возвращает его на фронтенд.
   - Бот строит план действий, выполняет команды пошагово, подтверждая выполнение каждого шага.
